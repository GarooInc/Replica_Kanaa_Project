# Utilities

## Propósito
Scripts auxiliares y complementarios para facilitar tareas recurrentes como manejo de archivos, procesamiento de imágenes, transformaciones o apoyo en procesos de data agent.

## Ejemplo principal
- **photo_uploader.py**: Permite subir y asociar la primera imagen identificada en el contexto o la respuesta del hotel. Este módulo es útil para agregar imágenes al Markdown de respuestas.

## Cómo usar/utilizar
Incluye ejemplos de uso, parámetros esperados y formatos de resultado para facilitar extensiones por otros desarrolladores.

## Mejoras posibles
Puedes ampliar esta carpeta incluyendo scripts de manejo de PDFs, exportaciones, validación de datos, etc.
